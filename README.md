
# Relax-IA Asistente de Dropshipping

Una aplicación web inteligente que simplifica la gestión de tu negocio de dropshipping utilizando IA.

## 🚀 Características

- **Análisis de Mercado con IA**: Identifica productos tendencia y nichos de mercado prometedores
- **Gestión de Proveedores**: Administra tus proveedores y productos de forma centralizada
- **Marketing Automatizado**: Genera contenido y copys para redes sociales con IA
- **Panel de Control Intuitivo**: Interfaz responsive y fácil de usar
- **Integración con Claude**: Potenciado por la IA de Anthropic para análisis avanzados

## 💻 Tecnologías

- Node.js
- Express.js
- EJS (JavaScript Incrustado)
- Claude AI (Anthropic)
- CSS3 con diseño responsive

## 🛠 Instalación

1. Clona el repositorio
2. Instala las dependencias:
```bash
npm install
```
3. Configura las variables de entorno en el panel de Secrets de Replit:
   - `ANTHROPIC_API_KEY`: sk-ant-api03-rlM...GQAA

## 🚀 Uso

1. Inicia el servidor:
```bash
npm start
```
2. Accede a la aplicación a través del enlace proporcionado por Replit
